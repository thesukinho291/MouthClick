# Security Policy

MouthClick pode mover e clicar com o mouse real. Use sempre em ambiente controlado e mantenha uma forma fisica de interromper o app.

## Regras de seguranca do projeto

- O app inicia bloqueado por padrao.
- `B` bloqueia/desbloqueia controle real e clique real.
- `ESC` desativa o controle real imediatamente.
- `Q` encerra o programa.
- PyAutoGUI `FAILSAFE` fica ativo.
- O app nao grava imagens ou videos da webcam.
- O app nao envia dados para servidores.

## Como reportar problema

Abra uma issue no GitHub descrevendo:

- sistema operacional;
- versao do Python;
- webcam usada;
- passos para reproduzir;
- se o problema envolve movimento ou clique inesperado.

Nao envie imagens pessoais da webcam em issues publicas.
