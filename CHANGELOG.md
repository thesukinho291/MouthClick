# Changelog

## 0.1.0

Versao inicial do MouthClick como prototipo funcional.

- Controle do cursor pelo nariz.
- Clique esquerdo por gesto de lingua.
- Botao unico para bloquear/desbloquear controle e clique real.
- Calibracao direcional do nariz.
- Calibracao do gesto de lingua.
- Configuracoes locais salvas.
- Logs locais sem imagem.
- Testes unitarios.
- Workflow de CI.
- Script de build com PyInstaller.
